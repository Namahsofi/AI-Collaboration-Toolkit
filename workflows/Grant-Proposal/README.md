# Grant Proposal Development ACW

## Purpose

Help nonprofit leaders develop high-quality grant proposals using the ACT Framework.

## Workflow

1. Discover
   - Understand the funding opportunity
   - Define project goals
   - Identify stakeholders

2. Prepare
   - Gather organizational information
   - Collect project details
   - Review funder requirements

3. Collaborate
   - Generate proposal drafts
   - Improve language
   - Develop supporting sections

4. Review
   - Verify facts
   - Check alignment with funder priorities
   - Validate completeness

5. Refine
   - Improve clarity
   - Strengthen storytelling
   - Finalize proposal

6. Reuse
   - Save reusable sections
   - Capture lessons learned
   - Create templates for future grants
